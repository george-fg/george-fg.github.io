What is "Scope"? and how does it work in JavaScript?

“Scope” refers to what code is visible from different parts of your entire program.

In JavaScript we have two scopes: global and local. When a variable is declared outside of a function, it’s defined as a global variable, and its value is accessible and modifiable throughout all of your code. A variable that is declared inside a function is local - it is created and destroyed every time a function is executed, and it cannot be accessed by any code outside of that function.
