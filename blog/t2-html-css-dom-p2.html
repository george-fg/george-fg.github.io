What is grid based design?


Grid based design is a way in which web developers can more easily plan and create a webpageâ€™s layout. Typically they create a grid that covers the entire page, made up of rows and columns. 12 columns per row is used, as 12 is the lowest number that can be divided by 2, 3 and 4. Once they have created this intersecting grid, they can put html content such as images or paragraphs into these intersecting grid squares.


Whats all the hype about responsive webpages?

Having responsive webpage design is important as we have multiple different screen sizes, such as tablets and smartphones. A good developer understands this and will make sure her webpage accommodates these different screen sizes. For example, when the page is being viewed below 300px, the layout will change. (This would be useful for smartphones!)


What is semantic structure?

Semantic structure refers to the semantic tags, and the order they are placed in. For example, we want the header tag at the top, followed usually by the nav, then they body, and so on. Good use of semantic structure allows both browsers and other programmers to more easily arrange and understand your code.
